文件权限

  • Android11 添加了分区存储的特性，在这个特性下，应用只能访问自身的应用专用缓存文件，比如相册的DCIM就无法访问。

  • 从 Android 11 开始，应用无法在外部存储设备上创建自己的应用专用目录。只能访问系统为应用提供的目录

  • 实测可以获取媒体文件Uri，如果使用媒体文件比如相片，就会将相片移动到应用文件夹再被打开。

  • 如何绕开？这个功能和IOS的沙箱机制非常相似，目前最方便的方案是将SDK设置为29，也就是退回Android10，再在application中添加一个配置忽略分区。
